### 1. 業務自動割当の段階的アルゴリズム改修

**要望**: 業務の自動割当ても段階を踏みたい
- 業務ごとに優先順位をつけたい（LD → 入浴 → 排泄 → フロア）
- できる業務が少ない人（専門スタッフ）を先に割り当て → 多能工は後から
- LDは最大1人まで

**対応**: `scheduler.ts` の `assignDuties` 関数を3段階アルゴリズムに書き換え

- **Step A**: 手入力済み業務のカウント
- **Step B**: 優先順位順に割当（ALL_DUTIES順: LD → 入浴 → 排泄 → フロア）、候補を `availableDuties.length` 昇順（専門スタッフ優先）でソート、不足数分だけ割当
- **Step C**: 残りスタッフへのフォールバック（フロアに割当）

**変更ファイル**: `src/lib/scheduler.ts`

### 3. シフト必要人数を絶対参照に変更

**要望**: シフト必要人数は絶対参照して

**対応**: Phase 5（残りスタッフの割当）を修正
- **変更前**: 公休ペースを優先し、必要人数が足りなくても休みにすることがあった
- **変更後**: 未充足のシフトがある場合、連勤上限・月間出勤上限に達していない限り**強制的にそのシフトに割当**。公休バランスより必要人数の充足を優先

```typescript
// 必要人数未充足のシフトがあれば強制割当（連勤上限を除く）
if (strictUnfilled && !hitConsecutiveLimit && !hitWorkLimit) {
  assign(s.id, date, strictUnfilled.id);
  // ...
  return;
}
```

**変更ファイル**: `src/lib/scheduler.ts`

### 5. 入浴も必要人数ピッタリに

**要望**: 入浴も必要人数ピッタリにして

**対応**: `assignDuties` の Step C を修正
- **変更前**: `withinCapDuty` で `req === 0` の場合も割当、フォールバックでは上限無視
- **変更後**: 全業務が必要人数に達したらフロアのみに割当。LD・入浴・排泄は必要人数ぴったりまでしか割り当てない

```typescript
// All capped duties are full — assign to floor (unlimited catch-all)
if (available.includes(FALLBACK_DUTY)) {
  assignments[idx] = { ...assignments[idx], duty: FALLBACK_DUTY };
  dutyCounts[FALLBACK_DUTY]++;
}
```

**変更ファイル**: `src/lib/scheduler.ts`

### 7. コード最適化の確認

**要望**: コード最適化とかする必要ある？

**回答**: 今すぐ最適化が必要な状態ではない
- `scheduler.ts` と `ShiftTablePage.tsx` に `find()` の多用あり → Map化で改善可能
- ただしデータ量が小さい（スタッフ数十人×31日）ので体感速度に影響なし
- 夜勤ペア組合せも上限12人で制限済み
- 機能追加やバグ修正を優先すべき

### 9. 月の合計勤務時間の追加

**要望**: 個人の月の合計勤務時間計算して公休の横へ（時間単位）。日勤帯は休憩60分引く、夜勤は90分引く

**対応**:
- `calcShiftHours` 関数を追加: startTime〜endTime の差分から休憩時間を差し引き
  - 日勤帯（isDayShift）: -60分
  - 夜勤（isNightShift）: -90分（日またぎ対応）
  - 明け/休み/有給: 0h
- 統計列に「時間」列を追加（ヘッダー + データセル）
- colSpan を 3 → 4 に調整

計算結果:
| シフト | 拘束 | 休憩 | 実働 |
|--------|------|------|------|
| L（早番）08:00-16:30 | 8.5h | -60分 | 7.5h |
| B（日勤）08:30-17:00 | 8.5h | -60分 | 7.5h |
| U（遅番）10:30-19:00 | 8.5h | -60分 | 7.5h |
| 〇（夜勤）16:30-09:00 | 16.5h | -90分 | 15h |
| 研（研修）09:00-17:00 | 8h | -60分 | 7h |

**変更ファイル**: `src/pages/ShiftTablePage.tsx`
