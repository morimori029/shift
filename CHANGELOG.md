# 更新履歴（CHANGELOG）

> シフト自動作成アプリの機能追加・修正の記録

---

## v2.0 — 2026-03-14

### 新機能

- **Undo（元に戻す）機能**
  - 最大10回まで操作を取り消し可能
  - ヘッダーに「元に戻す」ボタン表示（履歴があるときのみ）
  - Ctrl+Z（Mac: Cmd+Z）キーボードショートカット対応
  - フロア切替・月切替はUndoの対象外

- **自動バックアップ**
  - 5分間隔でブラウザ内に自動保存
  - サイドバーの「自動バックアップ保存」からダウンロード可能
  - 最終バックアップ時刻をサイドバー下部に表示

- **自動生成の失敗診断**
  - 生成後に人数不足が発生した場合、原因を診断表示
  - 対応可能スタッフ数・休み比率などの詳細情報を提供

- **前月夜勤の引き継ぎバナー**
  - 前月末に夜勤だったスタッフがいる場合、シフト表上部に琥珀色のバナーで通知
  - 「今すぐ明けを設定」ボタンで1日目の明けを一括設定

- **Excel 出力に業務表示**
  - 各スタッフを2行構成で出力（上段: シフト種別、下段: 業務名）
  - 業務名: LD・入浴・フロア・排泄・応援
  - シフト表下部に業務配置の日別集計セクションを追加

### バグ修正

- **localStorage 保存失敗時の通知**
  - ストレージ容量超過時にトースト通知で警告（以前は無言で失敗していた）

- **夜勤回数の min > max バリデーション**
  - nightShiftMin > nightShiftMax の矛盾設定を保存時に自動補正

- **祝日の警告表示修正**
  - 自動生成後の警告がスケジューラーの祝日ロジックと一致するよう修正
  - 祝日を日曜扱いで必要人数を判定

- **インポート時のクロスリファレンス検証**
  - assignments / pairSettings 内の staffId / shiftTypeId が実在するかチェック
  - 不整合データは自動除去し、警告メッセージを表示

- **インポート時のマイグレーション適用**
  - 旧フォーマットのデータに tags・availableDuties・unavailableDow 等を自動補完

- **シフト種別削除時のガード**
  - 削除後にスタッフの勤務可能シフトが空になる場合は削除を中止

### ドキュメント

- 仕様書を v2.0 に更新（データモデル・状態管理・Excel出力等を反映）
- マニュアルを v2.0 に更新（Undo・自動バックアップ・Excel業務表示の説明追加、FAQ 5件追加）

## v1.0 — 2026-03-04

### 初期リリース

- **スタッフ管理**
  - スタッフの登録・編集・削除
  - ドラッグ&ドロップで並び替え
  - 担当シフト・業務・出勤不可曜日の設定
  - 夜勤専従・短時間勤務フラグ

- **シフト設定**
  - フロアごとの曜日別必要人数設定（早番・日勤・遅番・夜勤）
  - 業務別必要人数設定（LD・入浴・フロア・排泄）
  - 祝日設定（全フロア共通）
  - 祝日専用必要人数設定
  - 最大連勤日数・公休日数・リーダー人数設定
  - 古いデータの一括削除機能

- **相性設定**
  - 夜勤ペアの NG / 推奨設定

- **シフト表**
  - シフトの自動生成（6フェーズ制約充足アルゴリズム）
  - セルクリックで手入力（ポップオーバー）
  - 夜勤入力時の翌日「明け」自動挿入
  - 手入力マーク（青ドット）表示
  - 「空欄を自動で埋める」「作り直す」「クリア」操作
  - 業務プルダウン（早番・日勤の下段）
  - 応援（onef）対応
  - 統計行（出勤日数・夜勤回数・公休日数）
  - 充足状況行（必要人数の過不足を色分け表示）
  - 日別コメント入力
  - キーボード操作（← → で月切替）
  - ヘッダー固定（Sticky）

- **シフト自動生成エンジン**
  - Phase 0: 夜勤→明け強制
  - Phase 0.5: 手入力の適用
  - Phase 1: 出勤不可曜日の強制休み
  - Phase 2: 夜勤配置（ペア相性・クールダウン考慮）
  - Phase 3: 日勤帯配置
  - Phase 4: リーダー配置
  - Phase 5: 未割当の処理（公休ペース調整）
  - Phase 6: 業務均等割り振り
  - 前月末の連勤数・夜勤引き継ぎ対応
  - 非常勤フロアの軽量ロジック

- **出力**
  - Excel 出力（.xlsx）
  - PDF 出力（A4 横向き印刷）

- **データ管理**
  - LocalStorage への自動保存・自動読み込み
  - JSON エクスポート / インポート
  - データマイグレーション（旧フォーマット互換）

- **技術**
  - React 18 + TypeScript + Vite
  - Tailwind CSS
  - vite-plugin-singlefile（dist/index.html 単体配布）
  - Vitest によるユニットテスト（6テスト）
